# Datasets

Download each dataset manually and place files in the paths shown below.

---

## CICIDS2017

Source: https://cicresearch.ca/CICDataset/CIC-IDS-2017/

Download **MachineLearningCVE.zip** from the CSVs folder and extract it.

Place the extracted files at:
```
data/CICIDS2017/MachineLearningCVE/
  Monday-WorkingHours.pcap_ISCX.csv
  Tuesday-WorkingHours.pcap_ISCX.csv
  Wednesday-workingHours.pcap_ISCX.csv
  Thursday-WorkingHours-Morning-WebAttacks.pcap_ISCX.csv
  Thursday-WorkingHours-Afternoon-Infilteration.pcap_ISCX.csv
  Friday-WorkingHours-Morning.pcap_ISCX.csv
  Friday-WorkingHours-Afternoon-PortScan.pcap_ISCX.csv
  Friday-WorkingHours-Afternoon-DDos.pcap_ISCX.csv
```

---

## NSL-KDD

Source: https://github.com/jmnwong/NSL-KDD-Dataset (original UNB link unavailable)

Download the repo as a zip and extract it.

Place the extracted files at:
```
data/NSL-KDD/
  KDDTrain+.txt
  KDDTrain+_20Percent.txt
  KDDTest+.txt
  KDDTest-21.txt
```

---

## UNSW-NB15

Source: https://cicresearch.ca/CICDataset/CIC-UNSW/

Download the four CSV files and the readme.

Place the extracted files at:
```
data/UNSW-NB15/
  CICFlowMeter_out.csv
  Data.csv
  Label.csv
  Readme.txt
```

Data sourced from: [UNB](https://cicresearch.ca/CICDataset/CIC-UNSW/browse.php?t=1772845468406).
