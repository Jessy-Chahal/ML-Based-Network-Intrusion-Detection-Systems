# ML-Based-Network-Intrusion-Detection-Systems
This project investigates the vulnerability of Machine Learning (ML)-based Network Intrusion Detection System (NIDS) to adversarial evasion attacks while implementing realistic, problem-space constraints. 

# Environment Setup

This project uses Conda to manage dependencies and ensure reproducibility across all machines.

## Installation Instructions

1. **Create the environment:**
   Run the following command from the root of the repository to download and install all pinned dependencies:
   
   
   `conda env create -f environment.yml`
